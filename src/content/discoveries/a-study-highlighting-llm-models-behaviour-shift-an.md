A study analyzing 2,430 real repository interactions reveals Claude Code’s tool selection patterns. It builds config systems instead of recommending LaunchDarkly, rolls custom JWT auth rather than using established libraries, yet decisively picks GitHub Actions (94%), Stripe (91%), and shadcn/ui (90%) when it does choose commercial tools.
